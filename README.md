# termz

Setup terminals on your own terms.

## Supported terminals/multiplexers

- Windows Terminal
- VSCode
- VSCodium

## Roadmaps

### V1 Roadmap
- Document `.termz` file format
- Support more terminals:
    - GNOME Terminal
    - Konsole
    - Kitty
    - iTerm

### V1.1 Roadmap
- Support multiplexers:
    - tmux

### V2 Roadmap

- Rewrite in Rust.
- Make `.termz` files executable.
- Publish on `winget`, `apt`, `apk`, etc.
